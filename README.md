# 🌐 Portfolio Website Tutorial

Source code for the **"Build a Stunning Portfolio Website"** tutorial on [Code with Kishor](https://www.youtube.com/channel/UCTBfWv_z17IGMb04emdV4cg) YouTube channel.

---

## 🎬 Watch the Tutorial

[![YouTube](https://img.shields.io/badge/YouTube-Watch%20Now-red?style=for-the-badge&logo=youtube)](https://youtu.be/Nw6PpUnBmYQ)

## ✨ Features

- Dark theme with cyan accent
- Glassmorphism navbar with CTA button
- Animated floating hero image
- Real tech logos using Devicon
- Project showcase with hover zoom effect
- Contact form with focus effects
- Fully responsive (mobile friendly)

---

## 🛠️ Built With

- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript
- [Google Fonts — Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk)
- [Devicon](https://devicon.dev/)

## 🚀 Getting Started

1. Clone the repo

```bash
git clone https://github.com/kkghosh01/portfolio-website-tutorial
```

2. Open `index.html` in your browser — done!

No installation, no dependencies.

---

## 🎨 Customization

| What to change | Where |
|---|---|
| Your name & title | `index.html` → Hero section |
| Your photo | `images/hero.png` & `images/about.png` |
| Your skills | `index.html` → Skills section |
| Your projects | `index.html` → Projects section |
| Your links | `index.html` → Contact section |
| Colors | `style.css` → `:root` variables |
